p. Matches documents that have fields that contain a term (*not analyzed*). The term query maps to Lucene @TermQuery@. The following matches documents where the user field contains the term @kimchy@:

<pre class="prettyprint lang-js">
{
    "term" : { "user" : "kimchy" }
}    
</pre>

p. A boost can also be associated with the query:

<pre class="prettyprint lang-js">
{
    "term" : { "user" : { "value" : "kimchy", "boost" : 2.0 } }
}    
</pre>

p. Or :

<pre class="prettyprint lang-js">
{
    "term" : { "user" : { "term" : "kimchy", "boost" : 2.0 } }
}    
</pre>
