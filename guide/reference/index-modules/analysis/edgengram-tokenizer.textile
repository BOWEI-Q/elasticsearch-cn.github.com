p. A tokenizer of type @edgeNGram@.

p. The following are settings that can be set for a @edgeNGram@ tokenizer type:

|_. Setting |_. Description |
|@min_gram@|Defaults to @1@.|
|@max_gram@|Defaults to @2@.|
|@side@|Either @front@ or @back@.|
